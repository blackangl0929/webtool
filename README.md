使用说明：
    根目录添加config.py，格式：
    #coding=utf8
    #调试
    DEBUG = True
    #后台路径
    ADMIN_PATH = 'xxx'
    #数据库
    SQLALCHEMY_DATABASE_URI = "mysql://xxx:xxx@localhost/webtool"
    SQLALCHEMY_TRACK_MODIFICATIONS = True
    #session 密钥
    SECRET_KEY = 'xxx'
